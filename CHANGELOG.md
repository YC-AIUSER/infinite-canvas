# CHANGELOG

## Unreleased

+ [新增] 画布空白区域支持双击打开节点选择菜单，并在点击位置创建节点。
+ [新增] Toonflow 资产卡新增「质感样板」：生成无人物六格材质微距参考，视频层只取表面纹理、磨损、颗粒与色调，并由分镜决策锁定表独立控制布光。
+ [修复] cano 视频生成请求增加 `Idempotency-Key`，同一次传输重试复用 Key，内容审核终态重提改用新 Key，并校验 multipart 与参考音频接收数。
+ [调整] Toonflow Module3 故事板改为不带参考图的 blockout 未贴图灰模，视频层改由故事板锁构图、角色卡锁外观、ST 色板锁颜色，并停用首帧组引用。
+ [调整] Toonflow 人物资产卡改为 16:9 横版人物设定页，固定展示正面、侧面、背面、面部特写、关键状态、装备细节与中文总述。
+ [优化] cano 视频任务遇到 `CONTENT_BLOCKED` 时按 3 秒起的间隔自动重新提交最多 2 次，其他失败仍立即返回原始错误。
+ [新增] Toonflow 资产库环节组化：资产卡不再闷在节点弹窗里，改为一个「资产库」组容器 + 每张卡一个真实画布子节点，铺开可见、可单独选中/拖动/重生成/删除；卡数据仍以资产库节点为唯一真相源，子节点为其派生视图，不影响级联/版本/导出。
+ [新增] Toonflow 多段环节组化：故事板页 / 首帧 / 视频工作台三个环节的段实例，从原 batch 分组升级为无限画布原生 Group 容器（环节头 + 各段实例为组内成员，整组可拖动）；段同步的建/归档/重排/连线逻辑与图片批量分组均不受影响。
+ [新增] 画布工具栏新增「一键整理」按钮：按连接流向从左到右分层重排全部节点（剧本→分镜→段实例→图→视频），父节点垂直居中对齐子块、同组多节点排成紧凑方阵、游离节点收到下方；一次撤销（Ctrl+Z）可整体还原，移动带平滑过渡。
+ [新增] Toonflow 生成节点时自动防碰撞：新建段实例落点自动避开画布上已有节点，撞上时向下轻挪到最近空位，不再压叠（保留原摆位规则，只在真重叠时避让）。
+ [新增] Toonflow 资产卡池新增「音频卡（人声参考）」：上传人声参考音频，接入视频生成的参考音频（火山 Seedance 直连生效；cano 待其音频接口修复后可用）。
+ [新增] Toonflow 生产画布「成片 / 导出」节点：汇总已通过的段视频，顺序连播预览、逐段下载、打包 ZIP（本期分段交付，不拼接单文件）。
+ [新增] Toonflow 生产画布「接缝检查」节点：相邻段视频并排预览 + 连续性人工勾选，全勾即通过、可跳过（不调 AI，按段对与视频版本记录已检状态）。
+ [新增] 本地 Agent（canvas-agent）注入 Toonflow 短剧方法论（原生多镜头直出、禁首尾帧续接），规划分镜 / 视频时守方法论、不自行建议首尾帧工作流。
+ [调整] Toonflow 画布恢复通用创建入口（文本/图片/视频/音频/生成配置/组/上传素材/我的素材）和拖线新建节点菜单，与普通画布保持一致。
+ [修复] 修复 Codex 插件打开画布后本地 Agent 未自动连接，并同步修正 Vite 启动指引。
+ [修复] 修复画布组节点遗留的多余类型判断导致的 TypeScript 构建报错。
+ [调整] canvas-agent 与 Codex 插件版本升级到 0.2.0，包含 site_navigate 等站点级工具和站点级连续会话。
+ [优化] 本地 Agent 连接说明明确区分插件 / 手动 MCP 才会增加 Codex token 消耗，直接运行 Agent 不受影响。
+ [优化] 优化本地 Agent 连接说明，区分 Codex 插件启动和直接运行 Agent 两种方式。
+ [新增] Agent 对话消息改用 streamdown 流式渲染，提升Markdown 内容展示效果。
+ [新增] Agent 新增画布、工作台、提示词库和素材等站点级工具。
+ [新增] Agent 面板改为全站常驻右侧栏，开关时同步推动顶栏和页面内容，并保留独立入口按钮。
+ [新增] Agent 新增 `site_navigate` 工具，支持页面跳转。
+ [新增] Agent 对话运行中支持一键停止，中断当前 Codex turn。
+ [调整] Codex 会话改为站点级连续线程，跨页面和跨画布保持同一上下文。
+ [调整] 移除仅前端调用 OpenAI responses 接口，统一走 MCP + 本地 Codex 链路。
+ [新增] 画布节点支持统一维护名称字段，默认显示在节点上方，并可直接双击名称编辑。
+ [新增] 画布新增组节点，支持节点拖入/拆出分组、拖拽高亮吸附和移动组时带动子节点。
+ [调整] 画布节点顶部工具条改为点击选中节点后显示，避免鼠标经过节点时频繁弹出。
